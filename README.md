# Chrysalis 🌱🌀

Chrysalis is a high-performance, cloud-native social simulation platform designed to create and study a virtual society of intelligent, LLM-powered agents.

# 🧠 About This Project

Chrysalis is an ​LLM-powered multi-agent system​ simulating emergent social dynamics through continuous cognitive evolution.  Drawing inspiration from the "evolutionary" sandbox concept of Spore, our agents aren't bound by predefined scripts. Instead, they learn, adapt, and evolve complex social behaviors—such as collaboration, social network formation, and knowledge transfer—from a set of fundamental rules.

### ​Core Metaphor:

Just as a caterpillar dissolves its body to become a butterfly, our agents periodically ​deconstruct and reformulate​ their cognitive architectures through:

• 🔄 Self-reflective learning cycles

• 🕸️ Dynamic social knowledge transfer

• 🧩 Spontaneous tool composition
